# Mockito
**Mockito** is an open source testing framework for Java released under the MIT License. The framework allows the creation of test double objects in automated unit tests for the purpose of test-driven development or behavior-driven development. Current version by **2024** year is **5.x**

## Reference Documentation
All you want to know about Mockito is hosted at [The Mockito Site](https://site.mockito.org/) which is [Open Source](https://github.com/mockito/mockito.github.io) .  
Enjoy Mockito!
